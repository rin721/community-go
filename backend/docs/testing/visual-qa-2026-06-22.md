## 发现与修复

| 问题 | 影响 | 处理 |
| --- | --- | --- |
| 移动端公开首页导航在 `390x844` 下横向裁切，`Log in` 和 `Platform console` 不完整 | 首屏导航不可读，影响移动端可用性 | 调整 `.console-nav` 小屏规则，改为可换行、居中、不拆词的导航按钮布局 |
| 初始化 owner 流程视觉截图中，`zh-CN` 默认界面仍展示 `Verify` 等英文 mock 文案 | 视觉证据与中文优先规则不一致，容易掩盖真实本地化问题 | 修正 Playwright setup schema mock，使用中文步骤标题、说明和字段标签，并重跑聚焦 e2e 与视觉截图 |
| 后台公告无写权限移动端截图只显示侧边导航，主内容被 `height: 100vh` 的侧栏压到首屏之外 | 窄屏后台首屏不可判断当前业务页面和权限状态，影响移动端可用性审查 | 在小屏规则中将 `.console-admin-sidebar` 改为自然高度，保留桌面粘性侧栏，重跑公告模块视觉截图 |
