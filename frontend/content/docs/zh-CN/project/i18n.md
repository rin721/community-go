# i18n

项目默认语言是 `zh-CN`，路由策略是 `no_prefix`。这意味着 `/docs/components/actions` 在三种语言中使用同一个路径，由当前 locale 决定渲染哪份内容。

## Locale 文件

共享用户可见文案维护在：

- `i18n/locales/zh-CN.json`
- `i18n/locales/en.json`
- `i18n/locales/ja.json`

新增共享文案时三份同步。部分既有功能页仍有内联中文，大幅触碰这些区域时优先迁移可复用文案。

## Docs 内容

Nuxt Content 定义了三个 collection：`docsZhCn`、`docsEn`、`docsJa`。三种语言保持相同目录结构与 slug，便于在不改变 URL 的情况下切换语言。

## 写作建议

文档标题、说明、导航名和代码注释应贴近目标语言，不要机械逐字翻译。代码标识、文件路径和组件名保持原文。
